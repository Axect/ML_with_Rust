5.1 Binary Variables

5.1.1 Bernoulli Distribution

제일 먼저 알아볼 확률 분포는 베르누이 분포(Bernoulli distribution)입니다. 베르누이 분포는 값을 두 개만 가질 수 있는 단일 확률변수에 대한 분포인데, 쉽게 말하면 동전 던지기입니다. 정의는 다음과 같습니다.

Definition 5.1 - Bernoulli distribution
Let consider a single binary random variable \(x\in\{0,1\}\). Suppose the probability of \(x=1\) is given as \(p(x=1|\mu) = \mu\) where \(0\leq \mu \leq 1\) then \[\text{Bern}(x|\mu) = \mu^x(1 - \mu)^{1-x}\] is called Bernoulli distribution.

앞에서 배웠던 기댓값과 분산의 정의를 이용하면 다음을 아주 쉽게 알아낼 수 있습니다.

Theorem 5.2 - Representative values of Bernouli distribution
Let \(\text{Bern}(x|\mu)\) be given Bernoulli distribution. Then \[\begin{align} \mathbb{E}[x] &= \mu \\ \text{var}[x] &= \mu(1-\mu) \end{align}\]
